## 技术栈

### 后端技术栈

- Spring Boot 3.2.0
- Spring Cloud 2023.0.0
- Spring Cloud Gateway
- Spring Cloud Netflix Eureka
- Spring Cloud OpenFeign
- MyBatis Plus 3.5.4.1
- Sa-Token 1.37.0
- MySQL

### 前端技术栈

- Vue 3
- Vue Router
- Pinia
- Element Plus
- Axios
- Vite

## 开发环境

- JDK 21
- Node.js 18+
- MySQL 8.0+
